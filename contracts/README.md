# Mina zkApp: Test

This template uses TypeScript.

## How to build

```sh
npm run build
```

# How to run test script

```sh
npx tsc && node build/src/run.js
```

## How to run tests

```sh
npm run test
npm run testw # watch mode
```

## How to run coverage

```sh
npm run coverage
```
